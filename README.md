# IOT - Assignement 1: Catch the Led Pattern!

This game was created as the assignement for the Embedded Systems and Internet of Things course.
The game is made to run on an Arduino Uno board. 

##  Components
- Arduino Uno board
- Breadboard
- Jumper wires
- 4 Green Leds
- 1 Red Led
- 1 Potentiometer
- 4 Buttons
- Resistors
